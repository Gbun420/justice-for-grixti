# Asset Setup Instructions

To complete the "Authentic Service" look of the landing page, you need to save a real photo of Dr. Silvio Grixti.

## 1. Get the Image

Since we cannot automatically download copyrighted news images, please follow these steps:

1. **Click this link** to view a professional photo from *Times of Malta*:
    [View Article & Photo](https://timesofmalta.com/article/silvio-grixti-among-charged-six-million-euro-benefit-fraud-scheme.1086036)
    *(Note: Look for the photo of him in a suit or entering a building. It conveys seriousness.)*

2. **OR** use this Google Search for more options:
    [Dr. Silvio Grixti Photos](https://www.google.com/search?q=Dr.+Silvio+Grixti+Malta+doctor+professional+photo&tbm=isch)

## 2. Save the File

1. Right-click the chosen image.
2. Select **"Save Image As..."**
3. Navigate to your project folder: `Desktop/untitled folder 2/public/`
4. **Important:** Name the file exactly **`grixti-service.jpg`**.

## 3. Refresh

Once saved, refresh your local site (`http://localhost:5173/`). The placeholder will automatically be replaced by the photo.
